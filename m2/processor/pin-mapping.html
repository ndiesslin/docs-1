Pin Mapping

Pin mapping

NOTE! This table is only applicable when using the "SamNonDuePin" library found here. We are working on a more M2-specific pin mapping that will not require any additional libraries. See development status here.

A "pin mapping" describes how the different circuits of M2 are connected to the processor. For example, the Green LED (DS6 on the schematic) is connected to pin 7 (AKA PA14) of the processor. To make things easier from a programming standpoint, we can map this to a pin name, in this case, the green LED is mapped to pin 23.

The following table builds on the pin mapping for the Arduino Due and is similar to the DUE mapping found here: https://www.arduino.cc/en/Hacking/PinMappingSAM3X
